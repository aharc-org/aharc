# Spacing Utilities Cheat Sheet (Oakwood v4)

## Semantic Spacing Tokens
--spacing-xs       → tiny
--spacing-sm       → small
--spacing-md       → paragraph rhythm
--spacing-lg       → content padding
--spacing-block    → block spacing
--spacing-section  → section spacing
--spacing-inline   → inline gap
--spacing-content  → content gap

------------------------------------------------------------
RESPONSIVE VARIANTS
------------------------------------------------------------

Breakpoints:
.sm → ≥480px
.md → ≥768px
.lg → ≥1024px

Pattern:
.u-mb-md.sm
.u-pt-lg.md
.u-ml-sm.lg
.u-stack-block.md

Examples:
.u-mb-md.sm     → margin-bottom: var(--spacing-md) at ≥480px
.u-pt-lg.md     → padding-top: var(--spacing-lg) at ≥768px
.u-stack-sm.lg  → stack spacing at ≥1024px
